    Parameters

    • Optional device: II2C

      The I2C device that represents the physical connection to the gyro. If null, then it is assumbed that the host is a revision 2 or higher board and a default I2CBus using the rev 2 I2C bus path will be used instead.

    • Default value busAddress: number = 0

      The bus address of the device.
