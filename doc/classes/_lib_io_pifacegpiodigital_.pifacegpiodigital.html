  • new PiFaceGpioDigital(pin: PiFacePins | null, initialValue?: PinState, spiAddress?: undefined | number, spiSpeed?: undefined | number, test?: undefined | false | true): PiFaceGpioDigital

  • Overrides PiFaceGpioBase.constructor

    • Defined in src/lib/IO/PiFaceGpioDigital.ts:117

    Initializes a new instance of the jsrpi.IO.PiFaceGpioDigital class with the pin to control, initial pin state, SPI address, and SPI speed.

    Parameters

    • pin: PiFacePins | null

      The PiFace pin to control.

    • Default value initialValue: PinState = PinState.LOW

      The initial value (state) to set the pin to. Default is PinState.LOW.

    • Optional spiAddress: undefined | number

      The SPI address to use. (Should be ADDRESS_0, ADDRESS_1, ADDRESS_2, or ADDRESS_3).

    • Optional spiSpeed: undefined | number

      The clock speed to set the bus to. Can be powers of 2 (500KHz minimum up to 32MHz maximum). If not specified, the default of SPI_SPEED (1MHz) will be used.

    • Optional test: undefined | false | true

    Returns PiFaceGpioDigital
