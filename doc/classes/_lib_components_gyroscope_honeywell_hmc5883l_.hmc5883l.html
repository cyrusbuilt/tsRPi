Class HMC5883L

classdesc

Represents a device abstraction component for a Honeywell HMC5883L 3-axis Digital Compass IC. See http://www51.honeywell.com/aero/common/documents/myaerospacecatalog-documents/Defense_Brochures-documents/HMC5883L_3-Axis_Digital_Compass_IC.pdf for details.
