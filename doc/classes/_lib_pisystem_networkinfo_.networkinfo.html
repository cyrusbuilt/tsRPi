Static getAllFQDNs

  • getAllFQDNs(): Promise<string[]>
    • Defined in src/lib/PiSystem/NetworkInfo.ts:71

    Gets all FQDNs of the machine. This enumerates all configured network addresses on all configured network interfaces, and translates them to DNS domain names. Addresses that cannot be translated (i.e. because they do not have an appropriate reverse DNS entry) are skipped. Note that different addresses may resolve to the same name, therefore the return value may contain duplicate entries. Do not make any assumptions about the order of the items in the array.

    Returns Promise<string[]>

    The FQDNs.
