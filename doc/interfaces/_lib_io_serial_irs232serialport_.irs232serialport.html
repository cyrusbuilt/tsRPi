Interface IRs232SerialPort

An interface that represents the RS-232 Serial port on the Raspberry Pi. IMPORTANT: You MUST explicitly call close() or dispose() when done with the port in order to release it. Even if the object instance is GC'ed, the port will NOT automatically be closed and therefore will remain locked even after program execution ends.
